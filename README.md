# VIEW-FUN-O-PL-pgSQL----PostgreSQL
O objetivo foi trabalhar com VIEW e função PL/pgSQL em cima de uma tabela de usuários.
O que foi feito

Criação da tabela users com os campos id, name, email e active
Inserção de usuários de exemplo
View active_users_view que retorna apenas os usuários ativos
Função check_user_status que recebe um id e retorna se o usuário é ativo, inativo ou não existe
